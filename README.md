# CapAndThief

우선 Google Gemini 로 만든 게임입니다.

📜 [경찰과 도둑: City Defense] 최종 게임 규칙서
지금까지 사용자님과 함께 완성한 최종 게임 시스템 명세입니다.

1. 목표 및 승패 조건
목표: 끊임없이 침입하는 도둑들을 체포하여 경찰서로 이송하고 최고 점수를 기록하세요.

패배(Game Over): 도시 내 도둑의 총인원이 30명을 초과하면 도시가 함락되어 패배합니다.

2. 플레이어 (경찰차)
기본 이동: W, A, S, D (기본 속도 80km/h)

부스트: Shift 누르고 이동. 스태미나를 소모하여 약 1.8배 빠르게 이동합니다.

무게 시스템 (페널티):

도둑을 체포할 때마다 차가 무거워집니다.

1명당 속도 감소. (많이 태우면 걷는 것보다 느려짐)

경찰서에 복귀하면 초기화됩니다.

위성 스캔: M 키를 누르면 언제든지 미니맵에 도둑의 위치(빨간 점)를 표시합니다.

3. 적 (도둑)
스폰(침입):

초기: 게임 시작 시 10초 간격으로 스폰.

가속: 점수 100점당 1초씩 단축됩니다. (최대 1초 간격까지 빨라짐)

진화: 시간이 지날수록, 많이 잡을수록 이동 속도가 점점 빨라집니다.

AI 행동:

배회: 도시 전체를 랜덤하게 돌아다닙니다.

도주: 플레이어가 접근하면 전력 질주하여 도망갑니다.

인구 제한: 중앙(경찰서) 구역에는 최대 5명까지만 머물 수 있습니다. (꽉 차면 외곽으로 튕겨 나감)

4. 아이템 시스템
생성: 게임 시작 1분(01:00) 후부터 생성됩니다.

종류 (랜덤):

❄️ TIME FREEZE (10초):

모든 도둑이 그 자리에 이동 불가 상태가 됩니다. (체포는 가능)

다음 도둑 침입 타이머가 멈춥니다.

화면이 푸르게 얼어붙는 효과가 나타나며 서서히 녹습니다.

🚧 SLOW AURA (30초):

플레이어 주변에 주황색 오라가 생성됩니다.

오라 범위 내의 도둑들은 속도가 60% 감소합니다.

5. 경찰서 (본부)
위치: 맵 정중앙 (World 1, 1).

기능 (진입 시 자동 수행):

범인 이송: 데리고 있던 범인을 수감하고 점수 획득.

정비: 무게 페널티 제거 & 부스트 게이지 100% 충전.

정보 갱신: 위성 스캔 1회 자동 발동.

6. UI 및 편의성
알림 로그: 아이템 획득, 체포, 스폰 경고 등의 메시지가 화면 중앙에 쌓이며, 배경이 있어 잘 보입니다.

계기판: 하단에 현재 속도(km/h)를 표시하는 아날로그/디지털 속도계가 있습니다.

미니맵: 도둑(빨강), 플레이어(초록), 경찰서(파랑), 아이템(하늘/주황) 위치 표시.

PAUSED: ESC로 일시정지 시, 우측에 조작법 가이드가 표시됩니다.
